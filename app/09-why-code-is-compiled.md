# Why Code Is Compiled

Many languages (C, C++, Rust, Go) must be compiled.

## Why?
- Computers understand machine code  
- Compilers translate human code  
- Compilation checks for errors  
- Compilation optimizes performance

GitHub stores source code.  
You clone → compile → run.
