# What GitHub Is

GitHub is a platform for hosting code, tracking changes, and collaborating with developers worldwide.  
It uses Git, a distributed version control system, to store every change made to a project.

## Key Concepts
- Repositories
- Commits
- Branches
- Pull Requests
- Issues
- Open Source
